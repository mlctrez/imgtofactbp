Factorio image to blueprint converter v0.1 instructions
clipboard component

Click on the blueprint book to select an image file or use Ctrl-V to paste an image from the clipboard. The book image will be replaced and a grayscale of the image will appear to the right.

Adjust the threshold slider and a preview image will appear to the right of the grayscale image. Black pixels will represent where the selected tile type will appear in the blueprint. Use the invert checkbox if needed based on your image.

When you are satisfied with the preview image, use the size checkboxes to select the widths to generate. Click on Copy and the blueprint string will now be copied into the clipboard.

Enjoy.
